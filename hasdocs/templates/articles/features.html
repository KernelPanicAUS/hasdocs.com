Features

Build automation

A post-commit hook is installed on GitHub repository. Documentations is rebuilt whenever you push new code. There is no need to upload and sync the docs.

Polyglotism

Various documentation generators are supported. Currently it supports Jekyll(HTML, Textile, Markdown, and Liquid), Sphinx(Python), and Javadoc(Java). RDoc and Pelican support will be added soon.

Cloud hosting

Built documentation is stored in Amazon S3. This means your documentation is always available from anywhere on the web. It is also highly scalable, reliable, and fast.

Private documentations

Limit access to the documentation at team or individual level. The permissions are synced with GitHub for easier management.
