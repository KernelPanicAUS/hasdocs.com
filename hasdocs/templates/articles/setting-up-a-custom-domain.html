Setting up a custom domain

Set the custom domain

You can set up a custom domain for your project at the project edit page. You can have multiple custom domains.

Set up DNS

You need to set up your DNS in order to custom domain names for your docs.

Top-level domain (TLD)

For a TLD like example.com, make your DNS A record to point to 111.111.111.111.

Subdomains

You can set a CNAME record pointing to your docs. For example, if your docs are served at user.hasdocs.com/example, create a cname record pointing to user.hasdocs.com.
