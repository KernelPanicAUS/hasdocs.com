Changing behavior for HasDocs

When HasDocs builds your project, HASDOCS environment variable is set to True. You can vary build behavior by this. In Sphinx's conf.py for example: {% pygmentize 'python' %} import os on_hasdocs = os.environ.get('HASDOCS', None) == 'True' if on_hasdocs: html_theme = 'default' else: html_theme = 'nature' {% endpygmentize %}
